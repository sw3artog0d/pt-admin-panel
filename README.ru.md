# Панель администратора физиотерапии (Версия на Flask)

Административная веб-панель для управления упражнениями по физиотерапии и мониторинга активных сессий пользователей.

Проект создан как учебное backend-приложение с использованием Flask и SQLite.
Он предоставляет функционал CRUD для управления упражнениями, аутентификацию администратора, пагинацию и контроль активных сессий.

## Функционал

* Система аутентификации для администраторов
* CRUD-операции для упражнений по физиотерапии
* Управление каталогом упражнений
* Мониторинг сессий пользователей
* Поддержка пагинации
* Интеграция с базой данных SQLite
* Интерфейс на базе Bootstrap

---

## Технологический стек

* **Язык:** Python 3
* **Фреймворк:** Flask
* **База данных:** SQLite3
* **Frontend:** HTML, CSS, Bootstrap
* **Управление окружением:** python-dotenv

## Структура проекта


## Установка и настройка

### 1. Клонирование репозитория

```bash
git clone <url_репозитория>
cd <имя_репозитория>
```

### 2. Создание виртуального окружения

```bash
python -m venv .venv
```

#### Windows

```bash
.venv\Scripts\activate
```

#### Linux / macOS

```bash
source .venv/bin/activate
```

---

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

---

### 4. Настройка переменных окружения

Создайте файл .env в корне проекта:

```env
SECRET_KEY=your-secret-key
ADMIN_USERNAME=admin
ADMIN_PASSWORD=admin123
```

---

### 5. Запуск приложения

```bash
python run.py
```
